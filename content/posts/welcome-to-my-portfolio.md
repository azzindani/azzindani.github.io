# Welcome!

This is a personal portfolio built as a **single-page web application**, hosted entirely on GitHub Pages.

## What You'll Find Here

- **Project write-ups** covering data analytics, machine learning, and AI
- **PDF presentations** viewable directly in the browser with a carousel viewer
- **Code walkthroughs** and tutorials

## How This Site Works

This portfolio works like a lightweight content management system:

1. **Posts** are written in Markdown and stored in the repository
2. **Images and PDFs** are uploaded to the `uploads/` directory
3. An **admin panel** lets you create, edit, and manage content using the GitHub API
4. Everything is **static** — no server required

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| HTML/CSS/JS | Frontend SPA |
| Marked.js | Markdown rendering |
| Highlight.js | Code syntax highlighting |
| PDF.js | PDF carousel viewer |
| GitHub API | Content management |
