# Sample PDF Presentation

This post demonstrates the PDF carousel viewer. When you navigate to this post, it opens the PDF in a LinkedIn-style carousel where you can:

- Navigate slides with arrow keys or buttons
- Swipe on mobile
- Click thumbnails to jump to any slide
- Download the PDF

## How to Add Your Own PDF

1. Go to the **Admin Panel** > **Upload Files**
2. Upload your PDF to the `uploads/pdfs/` directory
3. Create a new post with type set to **PDF Presentation**
4. Set the PDF path to your uploaded file

That's it! Your PDF will be viewable in the carousel viewer.
